# Hotel-Management

A Java-based hotel management system built with PostgreSQL and JDBC that supports core hotel operations. The system implements full CRUD functionality for customers, rooms, bookings, maintenance, and staff assignments, with role-based access for administrators, employees, and customers. It features validated SQL queries, exception handling, and date-based reporting to ensure reliable and accurate backend data management.
